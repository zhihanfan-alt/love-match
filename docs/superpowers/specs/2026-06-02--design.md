# Love Match - 爱心消消乐 游戏设计文档

## 项目概述

- **名称**：Love Match（爱心消消乐）
- **平台**：Web H5（手机浏览器）
- **风格**：可爱浪漫风 + 花卉元素
- **开发周期**：1-2个月
- **创建日期**：2026-06-02

## 技术栈

- **前端框架**：HTML5 Canvas + TypeScript
- **构建工具**：Vite
- **动画系统**：自研粒子系统
- **音效**：Web Audio API
- **部署**：纯静态文件

## 核心玩法

### 基本规则
- 简化版羊了个羊三消玩法
- 多层堆叠卡牌，点击3个相同图案消除
- 7个卡槽位置，满了游戏失败
- 10-15个关卡，难度递增

### 卡牌图案（8种）

| 图案 | 含义 | 彩蛋关联 |
|------|------|---------|
| 💕 爱心 | 爱情 | 爱心雨彩蛋 |
| 😘 Kiss | 亲吻 | Kiss烟花彩蛋 |
| 🌹 玫瑰 | 浪漫 | 道具增强效果 |
| 🌸 西府海棠 | 温柔 | 隐藏关卡触发 |
| ⭐ 星星 | 希望 | 分数加成 |
| 🌙 月亮 | 浪漫夜 | 夜间模式切换 |
| 🎁 礼物 | 惊喜 | 随机道具奖励 |
| 💎 宝石 | 珍贵 | 稀有度标识 |

## 彩蛋系统

### 1. 爱心雨 (HeartRain)
- **触发**：消除3张爱心牌
- **效果**：全屏飘落粉色爱心，背景渐变为粉色，播放浪漫音效
- **持续时间**：5秒

### 2. Kiss烟花 (KissFireworks)
- **触发**：消除3张kiss牌
- **效果**：
  - 第一阶段：全屏烟花绽放（3秒）
  - 第二阶段：爱心+玫瑰花瓣飘落（5秒）
  - 第三阶段：文字提示"爱你哟~"渐显
- **音效**：烟花音效 + 浪漫背景音乐

### 3. 玫瑰祝福 (RoseBlessing)
- **触发**：消除3张玫瑰牌
- **效果**：道具效果增强50%，持续30秒，屏幕边框出现玫瑰装饰
- **音效**：魔法音效

### 4. 海棠秘境 (BegoniaSecret)
- **触发**：消除3张西府海棠牌
- **效果**：进入特殊小游戏（记忆翻牌：翻出3对海棠花通关）
- **通关奖励**：全屏烟花 + 解锁海棠主题皮肤

## 音效系统

| 场景 | 音效 | 备注 |
|------|------|------|
| 背景音乐 | 轻柔钢琴曲 | 循环播放，可开关 |
| 点击卡牌 | 清脆"叮"声 | 每次点击 |
| 消除成功 | 欢快旋律 | 3连消除 |
| 彩蛋触发 | 专属BGM切换 | 每种彩蛋不同音乐 |
| 烟花效果 | 烟花爆炸声 | 连续绽放声 |
| 爱心飘落 | 轻柔风铃声 | 持续播放 |
| 道具使用 | 魔法音效 | 使用道具时 |
| 通关 | 胜利音乐 | 关卡完成 |
| 失败 | 温柔安慰音 | 卡槽满了 |

## 视觉设计

### 色彩方案
- **主色**：樱花粉 #FFB7C5
- **辅色**：浅紫 #DDA0DD、玫瑰红 #FF69B4
- **点缀**：海棠红 #FF4D6D、金色 #FFD700
- **背景**：粉色渐变 + 飘落花瓣粒子

### UI设计
- **卡牌**：圆角20px，带阴影，可爱图案
- **卡槽**：半透明毛玻璃效果
- **按钮**：圆角渐变，hover有爱心弹出
- **动画**：流畅的全屏动画过渡
